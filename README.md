# Fetch users -React

## Installation

1. **Clone the repository:**

   ```bash
   https://github.com/naveennr96/EastVantageUser.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd FetchUser_EastVantage
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```

4. **Start the application in development mode:**

   ```bash
   npm run dev
   ```
